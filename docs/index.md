# 訂餐系統
**「美訂美當」** 訂餐系統是由共同製作完成，現階段以服務台南一中學生為主

# 技術棧
前後端分離  
* 前端
    * Framework: React.js
    * Websockets
    
* 後端
    * Framework: Laravel 8
    * Authentication: Laravel Sanctum
